# KG

Project setup

**API**

Move to ```api``` directory
```
cd api
```

Copy ```.env.template``` file to ```.env``` and make your config
```
cp .env.template .env
```

Launch docker compose
```
docker-compose up
```

**MOBILE**

Move to ```mobile``` directory
```
cd mobile
```

Copy ```.env.template``` file to ```.env``` and make your config
```
cp .env-example .env
```

Install depedency
```
npm install
```

Launch expo app
```
npm start
```
